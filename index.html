Introducción

Se denomina red de datos a aquellas infraestructuras o redes de comunicación que se ha diseñado específicamente a la Transmición de información mediante el intercambio de datos.

Tipos de Redes

Hay diferentes tipos de redes informáticas que podemos utilizar. Cada una de ellas puede estar orientada a un tipo de usuario, a una necesidad concreta para conectar equipos.

  • Ver más

Capas del modelo OSI

Divide en siete capas el proceso de transmisión de la información, donde cada capa se encarga de ejecutar una determinada parte del proceso global de comunicación.

Contenido

Este módulo lo inicia en su camino hacia una carrera exitosa en Tecnología de la Información al brindarle una comprensión fundamental de la creación, operación y mantenimiento de redes.

Importancia

La comunicación es casi tan importante para nosotros como el aire, el agua, los alimentos y un lugar para vivir.

Tipos de redes

Hay redes de todo tamaño. Pueden ir desde redes simples, compuestas por dos PC, hasta redes que conectan millones de dispositivos.

Capas del modelo OSI

Referente como estándar internacional para las comunicaciones en red, al ofrecer un marco de trabajo conceptual que permite explicar el modo en que los datos se desplazan dentro de una red entre un dispositivo y otro.p>

Seguridad de redes

La seguridad de la red es una parte integral de la red informática, independientemente de si la red está en un hogar con una sola conexión a Internet o es una corporación con miles de usuarios.

Protocolos

para que los dispositivos finales puedan comunicarse a través de una red, cada dispositivo debe cumplir el mismo conjunto de reglas.

Asignacion de direcciones IPV4

Una dirección IPv4 es una dirección jerárquica de 32 bits que se compone de una porción de red y una porción de host.
